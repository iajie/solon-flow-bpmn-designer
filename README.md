## 什么是 Easy-Bpmn-Designer

Easy-Bpmn-Designer 是一个基于bpmn重新加工的流程设计器，它基于 Web Component，因此支持 Layui、Vue、React、Angular 等几乎任何前端框架。它适配了 PC Web
端~~和手机端~~，并提供了 亮色 和 暗色 两个主题。除此之外，它还提供了灵活的配置，开发者可以方便的使用其开发流程设计的应用。

更多关于 Easy-Bpmn-Designer，请访问官网： https://www.easy-flowable.online


## Easy-Bpmn-Designer 的与众不同之处

### 简单、友好、易用

Easy-Bpmn-Designer 基于 Web Component 开发，支持与任意主流的前端框架集成。Easy-Bpmn-Designer 使用更加友好的 LGPL 开源协议，通过 `npm i easy-bpmn-designer` 使用，
不用担心 GPL 协议可能带来的 GPL 传染问题。
