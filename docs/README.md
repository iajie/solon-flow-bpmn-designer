## Solon-Flow-Bpmn-Designer 的与众不同之处

### 简单、友好、易用

Solon-Flow-Bpmn-Designer 基于 Web Component 开发，支持与任意主流的前端框架集成。Solon-Flow-Bpmn-Designer 使用更加友好的 LGPL 开源协议，通过 `npm i solon-flow-bpmn-designer` 使用，
不用担心 GPL 协议可能带来的 GPL 传染问题。
